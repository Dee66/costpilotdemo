# Live Demo Interface

This directory contains the web-based interface for the CostPilot live demo.

**⚠️ This is a demonstration interface - not production software**

The `index.html` file provides:
- Interactive exploration of the demo scenario
- Visual representations of CostPilot findings
- No installation or setup required
- Hosted at https://dee66.github.io/costpilotdemo/

For the actual CostPilot product interface, see the separate product repository.
